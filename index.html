Q.1- "We are interested in retrieving only the students' names; all the names should be in caps.
            ['JOHN', 'BABA', 'YAGA', 'WICK']"

Ans:

        let studentRecords =
            [{name: "John", id: 123, marks: 98},
            {name: "Baba", id: 101, marks: 23},
            {name: "yaga", id: 200, marks: 45},
            {name: "wick", id: 115, marks: 75}];

        let res = studentRecords.map((item)=>item.name.toUpperCase());

        console.log(res);
Output: 
        [ 'JOHN', 'BABA', 'YAGA', 'WICK' ]

Explanation: In this question we have use the map function for get the output student name in capital form.
